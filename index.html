  • Worth a Shot

    For this project, my team wanted to tackle a common problem when you find yourself with a free night, you want to go out but you don't want to do the same old thing. Instead, with the help of Worth a Shot the user can mix things up when they get a mixer. The user can search on the application using any ingredient (e.g. rum, vodka, or orange juice) and get back six randomized drinks containing that ingredient, including a picture and a list of the other ingredients. Any that strike their interests, the user can then save. The user can then search their local areas for bar recommendations and save those as well. The user can then access their favorites by simply clicking the link in the top right.

  • Reel Good Movies

    The application has the functionality to search for movies and pull them from the OMDB api and shows various statistics of the movie as well as other users' reviews of the movie. The user may then log in to add the movie to their favorites list or to review the movie. The user may also click on a reviewer's username to visit their profile to see their other favorite movies and reviews or to follow them. Once logged in the user can also view their own profile to see their full list of favorite movies and all reviews they have written.


    This project includes full login/logout functionality and includes a well-organized database and server routing, in addition to a fleshed out front end utilizing handlebars that is fully responsive, including for mobile.
